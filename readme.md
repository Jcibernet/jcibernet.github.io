# juancampias.com

Personal site & portfolio for Juan Campias.
Live: **https://www.juancampias.com**

Static site hosted on **GitHub Pages** from the `master` branch. Spanish (`/es/`)
and English (`/`) versions are served from separate URLs for SEO.

---

## Stack

- HTML5 (two language entry points: `index.html`, `es/index.html`)
- **SCSS** (Dart Sass) → compiled to a single `styles/styles.css`
- Vanilla JS (`app.js`) — no framework
- Self-hosted fonts: **Space Grotesk** (variable) + **Poppins** subset
- Inline SVG icon sprite — no FontAwesome / icon CDNs
- Cache-busting via content hashes appended as `?v=<hash>` query strings

## Local development

Requires **Node 18+** and **Python 3** (for the cache-bust script).

```bash
# one-time
npm install

# watch mode — recompiles styles.css on every .scss save
npm run watch:css

# in another terminal — serve the static site
python3 -m http.server 8765
# open http://localhost:8765/
```

### Available scripts

| Command              | What it does                                                      |
| -------------------- | ----------------------------------------------------------------- |
| `npm run build:css`  | Compile `styles/styles.scss` → `styles/styles.css` (expanded)     |
| `npm run watch:css`  | Same, in watch mode                                               |
| `npm run build`      | `build:css` + run `tools/cache-bust.py` to refresh `?v=` hashes   |

### Pre-commit hook

`tools/hooks/pre-commit` is run on every commit. It:

1. Recompiles `styles.scss → styles.css` if any `.scss` file is staged.
2. Runs `cache-bust.py` so that `<link>` / `<script>` / `@font-face` URLs in
   `index.html`, `es/index.html` and `styles.css` always carry an up-to-date
   content hash. No Cloudflare purge needed.

To enable it locally:

```bash
git config core.hooksPath tools/hooks
```

To bypass it intentionally: `git commit --no-verify`.

## Editing styles

**Always edit `styles/styles.scss`.** `styles/styles.css` is regenerated and
should never be hand-edited. The pre-commit hook will recompile and re-stage
it for you, but during development run `npm run watch:css` so the browser sees
changes immediately.

The hero section uses the **"Circuit Fracture · Photo B&W"** design — a
duotone-tinted silhouette with glitch slabs and an inline-SVG circuit deco
layer. Mobile (≤768px) hides the slabs and re-anchors the photo to the top
half because they were sized for the desktop figure box.

## Deploy

Push to `master` and GitHub Pages publishes within ~1 minute. Workflow:

```
feature work → dev → PR → merge to master → live
```

The `dev` branch is the integration branch for in-progress work.

## Visual identity

Locked palette and typography are documented in
`.factory/skills/site-visual-identity/SKILL.md`. In short:

- Navy `#171d31` (primary background)
- Green `#055821` (accent / CTA)
- Grey ramp `#dbe1e8` → `#12181b`
- Space Grotesk (headings + body), Poppins as fallback
- No external icon fonts — only the inline SVG sprite
